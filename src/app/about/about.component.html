About Fashion MNIST Classifier

This project is an interactive web frontend for exploring and classifying images from the Fashion MNIST dataset. It provides a user-friendly interface to visualize, test, and understand machine learning models for image classification.

Project Purpose

The goal is to make machine learning more accessible by allowing users to interact with a neural network classifier trained on Fashion MNIST; a dataset of 70,000 grayscale images of clothing items. Users can upload images, view predictions, and learn how AI models distinguish between different fashion categories.

Key Features

  • Visualize sample images from the Fashion MNIST dataset.
  • Classify uploaded images and see real-time predictions.
  • Explore model confidence scores and prediction details.
  • Switch between light and dark modes for better accessibility.
  • Responsive design for desktop and mobile devices.

Technology Stack

  • Angular for building a reactive, modular frontend.
  • TailwindCSS 4 for modern, utility-first styling.
  • REST API integration for communicating with the backend classifier.
  • TypeScript for type-safe development.

How to Use

  1. Navigate to the Classifier page.
  2. Upload a fashion item image or select a sample.
  3. View the predicted category.
